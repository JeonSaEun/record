* Eclipse 화면개발 유형
(1) Static Web Project		: 정적인 문서만을 제공할 때 사용하는 프로젝트
	---> HTML5 (HTML, CSS, JavaScript, JQuery)
(2) Dynamic Web Project		: 동적인 문서를 제공할 때 사용하는 프로젝트
	---> JSP, Servlet을 이용해 개발
(3) Spring Legacy Project	: 사용중

* Modal: 가장 전면부에 표시하고, 뒤에있는 화면은 선택하지 못하게 하는 창
* Modaless: Modal창의 반대
